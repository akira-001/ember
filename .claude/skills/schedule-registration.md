# スケジュール写真からのカレンダー登録スキル

## トリガー
- Akiraが業務スケジュールの写真を送ってきた時
- カレンダー登録を依頼された時

## 手順
1. 画像を読み取ってイベント情報を抽出する
2. 以下の情報を特定する:
   - イベント名
   - 日時（開始・終了）
   - 場所（あれば）
   - 参加者（あれば）
3. `gcalcli`でAkira_publicカレンダーに登録する
4. 登録結果を一覧で報告する

## ルール
- **昼食・夕食の予定はカレンダー登録しない**
- カレンダー名は`Akira_public`を使う
- タイムゾーンはAsia/Tokyo
- gcalcliコマンド例: `gcalcli --calendar "Akira_public" add --title "xxx" --when "2026-03-25 10:00" --duration 60`
